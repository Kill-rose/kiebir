同階層の.exeをダブルクリックで起動します。


―　２．ゲーム概要・注意　―

目安プレイ時間　0.5h

・身内をモデルにしたキャラクター（身内しかわからないギャグはほとんどない）


―　３．操作方法　―

マウス専用ですが、一部のキー操作が対応しています（下記）。

●マウス操作
左クリック＆右クリック＆ホイール下　メッセージ送り
中クリック　メニューを開く
ホイール上　バックログを開く

●キー操作
Space＆Enter　メッセージ送り
S　セーブ画面を開く
L　ロード画面を開く
B　バックログを開く
M　メニューを開く
C　コンフィグを開く
A　メッセージ自動送りの切替
Ctrl　文章のスキップ（押している間）

●画面上のボタン
ITEM 　アイテム画面を開く
LOG  　バックログを開く
SAVE 　セーブ画面を開く
LOAD 　ロード画面を開く
AUTO 　メッセージ自動送りの切替
SKIP 　文章のスキップの切替
SETUP　設定画面を開く
TITLE　タイトルへ戻る
